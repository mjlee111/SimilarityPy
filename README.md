# Similarity Tester Python

## Installation
```shell
$ git clone https://github.com/mjlee111/SimilarityPy.git
$ cd SimilarityPy
$ pip install -r requirements.txt
```

## How To Use
### Txt file Test
```shell
$ cd SimilarityPy
$ mkdir files # Put your test files here
$ cd scripts
$ python3 txt.py
```
### Pdf file Test
```shell
$ cd SimilarityPy
$ mkdir files # Put your test files here
$ cd scripts
$ python3 pdf.py
```
### PyQt5 GUI Pdf Test
File select available with GUI
```shell
$ cd SimilarityPy
$ cd scripts
$ python3 qt.py
```
